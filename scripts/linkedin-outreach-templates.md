# Templates Outreach LinkedIn — Klevano
## Scripts de prospection pour gérants de commerces locaux

---

## CONTEXTE D'UTILISATION

**Cible :** Gérants / propriétaires de commerces locaux sur LinkedIn
**Volume :** Max 20 demandes de connexion/jour (limite LinkedIn)
**Stratégie :** Connexion → Message valeur → Relance douce
**Outil gratuit :** LinkedIn natif (pas besoin de Sales Navigator)

---

## RECHERCHE DES PROSPECTS SUR LINKEDIN

**Filtres de recherche :**
- Titre : "gérant", "propriétaire", "directeur", "fondateur", "restaurateur", "dirigeant"
- Localisation : ta ville cible
- Secteur : Restaurants, Coiffure, Artisanat, Commerce de détail, Santé

**URL de recherche type (copier-coller dans LinkedIn) :**
`site:linkedin.com/in gérant restaurant Bordeaux`

## MESSAGE 1 — Demande de connexion (160 caractères max)

### Version A (directe)
```
Bonjour {{Prénom}}, je travaille sur la visibilité Google des commerces locaux à {{Ville}}.
Votre profil m'a interpellé — heureux de vous rejoindre dans mon réseau.
```

### Version B (compliment)
```
Bonjour {{Prénom}}, {{NomEtablissement}} a l'air d'une belle adresse !
Je travaille sur le référencement local — ravi de vous connecter.
```

### Version C (curiosité)
```
Bonjour {{Prénom}}, j'analyse les fiches Google des commerces de {{Ville}} —
j'ai une info sur {{NomEtablissement}} qui pourrait vous intéresser.
```

## MESSAGE 2 — Premier message post-connexion (à envoyer dans les 24h)

```
Bonjour {{Prénom}},

Merci pour la connexion !

Je me permets d'être direct : j'ai regardé la fiche Google de {{NomEtablissement}}
et j'ai remarqué que vous avez {{NombreAvis}} avis. C'est bien, mais vos
concurrents directs sur Google Maps en ont souvent le double.

Concrètement, ça se traduit par des clients qui choisissent la concurrence
alors qu'ils auraient probablement préféré votre établissement.

Klevano automatise la collecte d'avis et l'optimisation de votre fiche Google —
nos clients gagnent en moyenne 2 positions sur Maps en 2 mois.

Est-ce que vous seriez ouvert à un appel de 15 minutes cette semaine pour
vous montrer ce que ça donnerait pour {{NomEtablissement}} ?

Pas de pression — juste pour voir si ça peut vous aider.

Bonne journée,
Thomas
```

## MESSAGE 3 — Relance si pas de réponse après 5 jours

```
Bonjour {{Prénom}},

Je reviens rapidement vers vous.

J'ai fait un audit express de votre fiche Google — voilà ce que j'ai noté :

• {{NombreAvis}} avis (moyenne du secteur à {{Ville}} : ~{{Moyenne}})
• Dernière activité sur la fiche : [à remplir manuellement]
• Taux de réponse aux avis : [à estimer]

Ces éléments expliquent probablement votre position actuelle sur Google Maps.

Si vous voulez voir comment l'améliorer rapidement, je suis disponible
pour 15 minutes en visio cette semaine.

Sinon, pas de problème — bonne continuation à vous !

Thomas — klevano.com
```

## OPTIMISATION DE VOTRE PROFIL LINKEDIN (avant de prospecter)

Ton profil doit être crédible pour maximiser le taux d'acceptation :
1. **Photo professionnelle** (pas un logo, une vraie photo de toi)
2. **Titre** : "Fondateur de Klevano — Référencement local Google My Business"
3. **Résumé** : Mentionner klevano.com + résultats clients (ex: "+2 positions sur Maps en 60 jours")
4. **Bannière** : Logo Klevano avec slogan
5. **Lien dans le profil** : klevano.com
